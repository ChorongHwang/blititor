Template Engine

Jinja2 스타일의 노드JS 템플릿 엔진 Nunjucks Template Engine 을 사용해 HTML 페이지를 수정할 수 있습니다.

Easy Customize

적절하게 즉시 사용할 수 있는 다양한 템플릿을 통해 원하는 디자인을 쉽게 만들어 낼 수 있습니다.

Module System

가장 강력한 모듈 생태계를 가지고 있는 npm 을 통해 확장 가능합니다.

Plays Nice

웹 개발자가 아니어도 따라하기를 통해 쉽게 운용이 가능합니다.

B L I T I T O R

플래인 테마를 사용하고 있는 블리티터 입니다.
블리티터의 팀블로그 모듈을 중심으로 꾸며진 예제로 회원관리, 블로그 포스트 관리, 방명록 관리 등의 기능을 제공하고 있습니다.
